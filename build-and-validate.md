## Disciplines every build increment owes (non-negotiable)
- **Unit / never-pool (┬з13.5):** no forced common scale; the analyzer must not pool across scales (the parity gate already locks secondary-axis exclusion тАФ extend, don't violate).
- **Censoring (┬з13.2):** `never` cost-of-virtue responses stay right-censored; never made finite (the |8.0| lock is the pattern).
- **No composite score:** never sum a branch's sub-indices into one number; report the facets.
- **Value-neutral / descriptive-only:** the reveal names where you sit; it never ranks. (Extra force for the charged branch, R6.)
- **N=1 interpretability:** the within-person reveal must be defensible for a single user, not just cohort-level.
- **Fraud / non-replication exclusions:** no Stapel / Gino / Ariely-priming / ego-depletion; R7's licensing (if/when built) scoped to the surviving half тАФ no Macbeth cleansing.

## Guardrails for the autonomous build loop
- **Never commit or push a red state.** Run `make check` before every commit; if red, fix or revert тАФ never push broken. (The design loop pushed prose; this loop pushes code to a public repo + a live runtime тАФ the bar is higher.)
- **Keep JSтЖФPython parity.** Anything touching the on-device reveal changes **both** `analyze.py` and `poc-projection.js`, and `check_impl_parity.py` must stay green.
- **Synthetic data only.** Fixtures with known ground truth; no real human data.
- **Propose, don't lock.** Pre-registration decisions are proposed for Dave's lock, never auto-locked (same as the design loop).
- **One branch per iteration; stop when dry.** If a scan finds no buildable-without-Dave branch left, journal it and halt тАФ don't pad.
