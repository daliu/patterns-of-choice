# Literature Index

Notes from the first literature review pass on `concept.md` (draft 0.1).

This index catalogs reviewed sources by area (the first eight files are the draft-0.1 pass; `narrative-immersion.md` was added later as the H8 theoretical anchor). Each file follows the format:
- Key sources (author, year, title)
- 2–4 sentences on what each contributes
- Note on relevance / contradiction with the concept doc

## Coverage status

Initial pass; aim was ~50–80 high-quality references. Density is uneven:
- Strong: measurement, behavior-change interventions, replication
- Medium: cross-cultural, moral psychology core
- Light: adjacent instruments (the commercial space is murky), ethical frameworks (philosophical literature is vast — surfaced only directly-relevant entries)

## Open lines for future passes

- Deeper read on Atari et al. 2023 MFQ-2 measurement invariance — the 49-culture Schwartz PVQ-RR and 25-culture MFQ-2 datasets are jointly the cleanest cross-cultural foundation available
- Meta-analyses on contemplative-practice apps (Headspace, Calm) for retention / adherence base rates — the comparison case for "what kind of attrition can a serious daily practice expect"
- The "wise interventions" handbook (Walton, Crum, Yeager 2023) — full survey of brief identity-belonging-purpose interventions; concept doc would benefit from referencing this catalog directly
- Behavioral game-theory literature on cooperation / signaling beyond the canonical economic games — relevant for the cost-of-virtue probe design
- Self-control and ego-depletion: the field has been heavily revised post-Hagger et al. 2016 failed replication; concept doc should not lean on it
